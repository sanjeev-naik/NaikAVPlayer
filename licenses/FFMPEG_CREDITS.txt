=========================================

This package distributes precompiled shared library (.dll) files for FFmpeg:
- avcodec-61.dll
- avformat-61.dll
- avutil-58.dll
- swscale-7.dll
- swresample-4.dll

`licenses/LICENSE.lgpl-2.1`.

The original, unmodified source code for FFmpeg used to build these binaries is
available from:
- FFmpeg Official Website: https://ffmpeg.org
- Source Downloads: https://ffmpeg.org/download.html
- Git Repository: https://github.com/FFmpeg/FFmpeg

No modifications were made to the FFmpeg source code. The binaries were built
using standard configure flags for LGPL distribution (such as disabling non-free/GPL
components).

dynamic libraries with your own builds of the libraries, provided they maintain
